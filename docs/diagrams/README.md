# توثيق مشروع النظام المصرفي الآمن

## مقدمة

يحتوي هذا المجلد على التوثيق الكامل لمشروع النظام المصرفي الآمن، الذي تم تطويره باستخدام لغة Python وإطار Flask.

تشمل الوثائق شرح فكرة المشروع، تصميم قاعدة البيانات، الخصائص الأمنية، الاختبارات، دليل الاستخدام، المخططات، وصور واجهات النظام.

## التقنيات المستخدمة

| التقنية | الاستخدام |
|---|---|
| Python | لغة البرمجة |
| Flask | إطار تطوير تطبيق الويب |

| Pytest | الاختبارات الآلية |

| Markdown | كتابة التوثيق |

## طريقة عرض الملفات

يمكن فتح أي ملف Markdown في Visual Studio Code، ثم استخدام:

Ctrl + K ثم V
لعرض المعاينة بجانب الملف.

يمكن أيضًا استخدام:

Ctrl + Shift + V
لفتح المعاينة في نافذة كاملة.

يفضل تثبيت إضافة:

Markdown Preview Enhanced
لعرض المخططات والجداول بطريقة احترافية.
